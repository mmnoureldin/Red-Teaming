Infrastructue

This post is about how to install a red team basic infrastructure using Apache and C2 framework from BishopFox. It is meant as the kickoff post for a series of tutorial posts on how to red teaming excercises from my experiments.

Setup Apache Redirector

Redirectors are used as a protection for team server, it shouldn't reach team server directly in anyway. Traffic from victim corporate should land on the redirector then, redirector will decided to forward it as per the configured rules.

In our case we are going to use Apache web server as the redirector, moreover we will use its' redirection rules to forward our C2 connections.

  1. In real life scenarios, we should have a public server with valid certificate and domain name, it may also host any fake web site.
